# ⚽ World Cup 2026 Shootout

A fast-paced, interactive **penalty shootout typing game** where you name World Cup players to score goals. One wrong answer ends the game — sudden death mode. Test your knowledge of international football superstars!

## 🎮 How to Play

1. **Start the shootout** – Click "Start Shootout" on the home screen
2. **Read the clue** – Each of 10 kicks presents a flag 🚩 and a player description
3. **Type the name** – Enter the player's name in the text field
4. **Take your shot** – Hit the "Kick ⚽" button to submit
5. **GOAL or MISS**
   - ✅ **Correct** → Ball flies into the net, confetti, crowd roars, move to next kick
   - ❌ **Wrong** → Keeper dives, ball sails wide, shootout over immediately

## 🎯 Features

- **10 progressive kicks**: Starts easy (Messi, Ronaldo) → medium (Haaland, Vinícius Jr) → hard (Yamal, Modrić) → extreme (Alphonso Davies)
- **Smart matching**: Answer matching is forgiving—type `mbappe` and it accepts "Mbappé", `RONALDO` matches any case/accent variant
- **Mobile-first design**: Built for smartphones (375×812 viewport), fully responsive
- **Live audio synthesis**: Goal cheers and kick sounds generated via Web Audio API—no external files
- **Embedded fonts**: Baloo 2 & Nunito fonts baked into the HTML—zero dependencies
- **Self-contained**: Single HTML file, ~148 KB, zero server requirements
- **Smooth animations**: Soccer ball trajectory, keeper dives, confetti bursts, stage shake effects
- **Perfect-game celebration**: Score all 10 and see the "PERFECT SHOOTOUT!" screen with triple confetti

## 🔊 Audio

All sound effects are **synthesized in real-time**:
- ⚽ **Kick**: Low frequency thump + sub-bass
- ✨ **Cheer**: Rising tone glissando (4-note sequence)
- ❌ **Buzz**: Buzzer on miss
- 🔔 **Whistle**: Final whistle, end-of-game

No MP3/WAV files—everything generated by oscillators and gain envelopes in Web Audio.
